# Geometry Dash HTML5 Conversion

Welcome to the Geometry Dash HTML5 conversion project! This project aims to convert the popular Geometry Dash game into an HTML5-based version, making it accessible directly through web browsers.

## Overview

Geometry Dash is a rhythm-based platformer game where players navigate through challenging levels filled with obstacles and musical beats. This HTML5 conversion brings the core gameplay experience to the web platform.

## Features

- **Browser-Based Gameplay**: Play directly in your web browser without any additional installations
- **Cross-Platform Compatibility**: Works on desktop, tablet, and mobile devices
- **Responsive Design**: Adapts to different screen sizes and resolutions
- **Level Editor Support**: Create and share custom levels
- **Sound and Music**: Full audio support with dynamic background music

## Getting Started

### Prerequisites

- Modern web browser with HTML5 support
- JavaScript enabled
- No additional dependencies required

### Installation

1. Clone the repository:
```bash
git clone https://github.com/Quantumshawn/exetohtm5.git
```

2. Navigate to the project directory:
```bash
cd exetohtm5
```

3. Open `index.html` in your web browser or serve the files using a local server:
```bash
python -m http.server 8000
# or
npx http-server
```

- **Arrow Keys / WASD**: Move and jump
- **Space**: Jump (alternative)
- **Mouse**: Click to interact with menus
- **R**: Restart level
